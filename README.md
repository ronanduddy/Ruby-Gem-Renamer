# Rename Gem

Description of project

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rename_gem'
```

And then execute:

```Shell
bundle install
```

Or install it yourself as:

```Shell
gem install rename_gem
```

## Usage

```Shell
bundle exec rename_gem . 'new_name'
```
